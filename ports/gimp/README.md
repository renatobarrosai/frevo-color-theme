# Frevo Palette for GIMP

A GIMP palette using the Frevo colors. Values are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation
1. Copy `frevo.gpl` to your GIMP palettes directory, e.g.:
   - Linux: `~/.config/GIMP/2.10/palettes/` (or `~/.gimp-2.8/palettes/` on older versions)
   - Windows: `%APPDATA%\GIMP\2.10\palettes\`
   - macOS: `~/Library/Application Support/GIMP/2.10/palettes/`
2. Restart GIMP and select the "Frevo" palette in the Palettes dialog.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
