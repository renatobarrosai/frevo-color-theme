# Frevo Colors for Calibre (viewer CSS)

A CSS snippet for Calibre's ebook viewer using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Files
- `frevo.css` — Calibre viewer stylesheet

## Installation
1. Open Calibre **Preferences → Look & Feel → Book viewer → Colors**.
2. Enable **User CSS** and paste the contents of `frevo.css`.
3. Save and reopen a book to see the theme.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- This targets the viewer content area; Calibre UI theming depends on your desktop theme.
