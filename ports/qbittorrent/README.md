# Frevo Colors for qBittorrent (Web UI CSS override)

A CSS override for qBittorrent Web UI using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation
1. Enable the alternative Web UI or inject custom CSS (e.g., via browser userstyle or modified webui directory).
2. Apply `frevo.css` to the Web UI assets or as a userstyle for the Web UI URL.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- qBittorrent UI structure can vary by version; selectors may need adjustment over time.
