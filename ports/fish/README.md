# Frevo Theme for fish

A vibrant dark fish shell theme using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation
1. Copy or symlink `frevo.fish` to your fish config directory, e.g.:
   - `~/.config/fish/themes/frevo.fish`
2. Load the theme in `config.fish`:
   ```fish
   source ~/.config/fish/themes/frevo.fish
   ```

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
