# Frevo Theme for Kate/KWrite

A vibrant dark Kate/KWrite color scheme using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Files
- `frevo.json` — Kate color scheme definition

## Installation
1. Copy `frevo.json` to your Kate color schemes directory, e.g.:
   - `~/.local/share/org.kde.syntax-highlighting/themes/frevo.json`
2. In Kate/KWrite, open **Settings → Color Themes** and select "Frevo".

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
