# Frevo Theme for GitHub

A vibrant dark GitHub theme using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Files
- `frevo.json` — GitHub theme definition

## Installation
1. Install via GitHub settings:
   - Go to **Settings → Appearance → Themes → Import theme**.
   - Paste the contents of `frevo.json` or select the file.
2. Choose the "Frevo" theme in your GitHub appearance settings.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
