# Frevo Theme for Rofi

A vibrant dark Rofi theme using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation
1. Copy or symlink `frevo.rasi` to your Rofi themes directory, e.g.:
   - `~/.config/rofi/frevo.rasi`
2. Launch Rofi with the theme or set it in your config:
   - `rofi -show drun -theme ~/.config/rofi/frevo.rasi`

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- Adjust layout (width, spacing, radius) as desired; colors remain unchanged.
