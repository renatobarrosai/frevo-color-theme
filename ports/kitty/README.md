# Frevo Theme for Kitty

A vibrant dark Kitty theme using the Frevo palette. Colors are sourced directly from `theme/frevo-theme.yaml` with no modifications.

- `frevo.conf` â€” Kitty color configuration

## Installation
1. Copy or symlink `frevo.conf` to your Kitty themes directory, e.g.:
   - `~/.config/kitty/themes/frevo.conf`
2. In your `kitty.conf`, include the theme file:
   - `include themes/frevo.conf`

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
