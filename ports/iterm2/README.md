# Frevo Theme for iTerm2

A vibrant dark iTerm2 theme using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Files
- `frevo.itermcolors` — iTerm2 color profile

## Installation
1. Open iTerm2 Preferences → Profiles → Colors.
2. Import `frevo.itermcolors` via "Color Presets..." → "Import".
3. Select "Frevo" from the presets list.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
