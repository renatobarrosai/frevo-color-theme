# Frevo Theme for YouTube Music (userstyle)

A userstyle for YouTube Music using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation (Stylus/Stylebot)
1. Install the Stylus (or Stylebot) browser extension.
2. Create a new style and paste the contents of `frevo.user.css`.
3. Apply it to `https://music.youtube.com/*`.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- YouTube Music DOM can change; selectors may need maintenance over time.
