# Frevo Palette for FreeCAD

A FreeCAD color preset using the Frevo palette. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Files
- `frevo.json` — FreeCAD color preferences

## Installation
1. Open FreeCAD and go to **Edit → Preferences → Display → 3D View**.
2. In the **Colors** tab, import `frevo.json` (Color preset).
3. Apply the preset to use Frevo colors in the 3D view and UI accents where applicable.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- Preset maps common UI/3D colors; adjust per workspace if needed.
