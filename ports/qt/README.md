# Frevo Theme for Qt (Kvantum)

A vibrant dark Kvantum theme palette using the Frevo colors. Values are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Installation
1. Copy `frevo-kvantum.kvconfig` to your Kvantum themes directory, e.g.:
   - `~/.config/Kvantum/Frevo/Frevo.kvconfig`
2. (Optional) Symlink or rename the file to match `Frevo.kvconfig` if needed.
3. Select the theme in `kvantummanager`.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
- This provides colors only; you can pair it with an existing Kvantum SVG theme or create one for full widget styling.
