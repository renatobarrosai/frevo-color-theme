# Frevo Palette for Figma

Frevo color tokens for Figma. Colors are taken directly from `theme/frevo-theme.yaml` with no modifications.

## Usage
1. In Figma, use the Tokens Studio (or similar) plugin.
2. Import `frevo.json` as color tokens.
3. Apply the "Frevo" color set to your styles.

## Notes
- Variant: dark
- Accent: blue
- Background avoids pure black to reduce OLED haloing.
